Настройки публикации
Прикрепленные файлы
Изменения в списке прикрепленных файлов не сохраняются пока вы не сохраните этот материал. Первый файл из перечисленных будет включен в ленты RSS.
Изображения больше чем 1024x1024 должны быть уменьшены Максимальный размер загружаемого файла 1 Мб. Можно загружать файлы только со следующими расширениями: jpg jpeg gif png txt doc xls pdf ppt pps odt ods odp swf flv.
Настройки адресов
Для вас будет сгенерирован синоним. Если вы хотите задать собственный синоним, отключите эту опцию.
Можно указать альтернативный адрес (синоним), по которому этот материал будет доступен. Например, укажите "about" для страницы "О компании". Используйте относительный путь без слеша в конце, иначе синоним работать не будет.
